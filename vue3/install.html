    # 2.环境搭建

    由于Vue CLI 的包名称由 vue-cli改成了@vue/cli，如果之前本地有安装vue-cli(1.x 或 2.x)的脚手架需要先进行卸载

    //卸载vue-cli(1.x 或 2.x)
npm uninstall vue-cli

//安装@vue/cl(3.x+)
npm i -g @vue/cli -g
